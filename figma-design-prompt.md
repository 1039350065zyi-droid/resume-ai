# ResumeAI — Figma 全站 UI 设计提示词

## 项目概述

ResumeAI 是一个 AI 驱动的智能简历优化平台，帮助求职者分析简历与岗位 JD 的匹配度、优化简历内容、生成面试题、并输出格式化简历文档。

目标用户：25-35 岁的职场求职者，互联网/科技行业为主。
品牌调性：**专业、智能、可信赖**，同时不过于严肃，带一点科技感和温度。

## 一、设计系统

### 1.1 配色方案

**主色（Primary）**

- 主色 Hover：#4338CA（Indigo 700）
- 主色 Light：#EEF2FF（Indigo 50）

- 成功/匹配：#10B981（Emerald 500）
- 警告/部分匹配：#F59E0B（Amber 500）
- 错误/缺失：#EF4444（Red 500）
- 信息：#3B82F6（Blue 500）

**中性色**
- 背景：#F8FAFC（Slate 50）
- 卡片：#FFFFFF
- 主文字：#0F172A（Slate 900）
- 副文字：#64748B（Slate 500）
- 占位符：#94A3B8（Slate 400）
- 边框：#E2E8F0（Slate 200）

- 全局背景使用微妙的渐变光晕效果：
  - 左上角 rgba(99,102,241,0.08) 椭圆
  - 右下角 rgba(168,85,247,0.06) 椭圆
  - 中心 rgba(14,165,233,0.04) 椭圆
  - 底色 #F8FAFC

### 1.2 字体

- 英文：SF Pro Display / Inter / system-ui
- 中文：Noto Sans SC / PingFang SC / Microsoft YaHei
- 等宽（代码/提示词）：JetBrains Mono / Fira Code

**字号规范**
- H1 大标题：40-48px / Bold 800
- H2 区域标题：20-24px / Bold 700
- H3 卡片标题：16-18px / Semibold 600
- 正文：14px / Regular 400
- 小字/说明：12px / Regular 400
- 标签/Badge：12px / Medium 500

### 1.3 间距与圆角

- 页面最大宽度：1024px（max-w-5xl）
- 页面左右内边距：24px
- 卡片内边距：24-32px
- 卡片间距：16-20px
- 组件间距：12-16px

- 圆角：
  - 大卡片/面板：20px
  - 小卡片/按钮：12px
  - Badge/标签：20px（全圆角）
  - 输入框：12px
  - 头像/图标容器：16-20px

### 1.4 阴影

- 默认阴影：0 1px 3px rgba(0,0,0,0.05), 0 1px 2px rgba(0,0,0,0.03)
- Hover 阴影：0 4px 12px rgba(0,0,0,0.07), 0 1px 3px rgba(0,0,0,0.05)
- 浮层阴影：0 10px 30px rgba(0,0,0,0.08), 0 2px 8px rgba(0,0,0,0.04)
- 主色按钮阴影：0 2px 8px rgba(79,70,229,0.3)

### 1.5 动画

- 页面入场：fadeUp（0.45s ease-out），各区域错开 0.08s
- 卡片 Hover：shadow 渐变（0.25s ease）
- 按钮点击：scale(0.97)
- 面板展开/收起：max-height 过渡（0.3s）
- 进度条：width 过渡（0.4s ease-out）
- 加载旋转：1s linear infinite

## 二、页面设计

### 2.1 主页 — 工作台（/）

这是核心页面，所有操作在一个页面内完成，不需要跳转。

#### 顶部导航栏
- 背景：白色毛玻璃（rgba(255,255,255,0.82)，backdrop-blur 20px）
- 固定在顶部（sticky）
- 左侧：Logo（渐变色方形图标 40x40，圆角 12px）+ "ResumeAI" 标题 + "智能简历优化平台" 副标题
- 右侧："⚙️ 后台管理" 文字按钮

#### Hero 区域
- 居中标题："让 AI 帮你打造完美简历"
  - "AI" 两个字使用渐变色（Indigo → Purple）
  - 字号 40-48px，Bold 800
- 副标题："上传简历和目标岗位JD，AI基于8大优化规则智能分析优化"
  - 字号 18px，Slate 500
- 上下留白充足（上方 64px，下方 56px）

#### 上传区域
- 两列并排（桌面端），单列堆叠（移动端）
- 每个上传卡片：
  - 白色实心卡片（card-solid），圆角 20px
  - 标题："📄 个人简历" / "📋 岗位 JD"（14px, Semibold）
  - 虚线边框拖拽区（border-dashed, Slate 200）
  - 拖拽区中央：渐变色图标容器（64x64，Indigo 50 → Purple 50 背景）
  - 提示文字："拖拽文件到此处或点击选择"
  - 支持格式说明："支持 PDF、TXT、MD、图片"
  - Hover 状态：边框变 Indigo 300，背景变 Indigo 50/30
  - 底部："或直接粘贴文字 ↓" 文字按钮
- 上传成功后：
  - 文件卡片（Emerald 50 背景，Emerald 100 边框）
  - 文件图标 + 文件名 + 文件大小
  - 右侧 ✕ 删除按钮

#### 进度条（分析中显示）
- 白色卡片，内含：
  - 左侧：旋转加载图标 + 步骤文字
  - 右侧：已用时长
  - 进度条：高度 8-10px，渐变色（Indigo → Purple），圆角全圆
  - 底部居中提示："AI分析约需1分钟，请耐心等待"
  - 百分比数字显示

#### 操作按钮区
- 居中主按钮："🚀 开始分析简历"
  - 渐变背景（Indigo → Purple），白色文字
  - 圆角 12px，padding 14px 40px
  - 阴影：0 2px 8px rgba(79,70,229,0.3)
  - Hover：上移 1px + 阴影加深
  - Disabled：opacity 0.4

#### 结果面板（可折叠）
每个结果面板是一个可折叠卡片：

**面板头部**
- 左侧：渐变色图标容器（48x48）+ 标题 + Badge（如"85 分"）
- 右侧：下载按钮 + 展开/收起箭头
- 收起时整个卡片可点击展开
- 展开时只有头部可点击收起

**匹配度分析面板（蓝色主题）**
- 环形分数图（170px，渐变色描边，数字居中）
- 四维度指标卡片（2x2 网格）：
  - 每个卡片：图标 + 标签 + 分数
  - 分数颜色：≥80 绿色渐变，≥60 靛蓝渐变，<60 橙色渐变
- 关键词分析区：
  - 左列：已匹配关键词（绿色 Badge）
  - 右列：缺失关键词（红色 Badge）
- 硬性条件检查：
  - 列表项：左侧需求+说明，右侧状态 Badge（达标绿/部分橙/未达标红）
- 优势/不足/建议三列：
  - 三个彩色卡片（绿/红/蓝），各含要点列表

**简历优化面板（绿色主题）**
- 模式选择：三个卡片（智能推荐/完整/自定义）
  - 选中态：蓝色边框 + 阴影
- 优化中：旋转动画 + 进度条
- 结果展示：
  - 已应用规则 Badge 列表
  - 优化后简历文本区
  - 修改详情：每个修改项显示原文（红底）vs 优化后（绿底）

**面试题面板（紫色主题）**
- 行为面试题 + 技术面试题分组
- 每组：图标 + 标题 + 题目数量 Badge
- 每题卡片：
  - 题号 + 考察要点 Badge + 题目
  - 展开后：回答技巧 + 参考答案
  - 展开/收起动画

**生成简历面板（绿色主题，独立面板）**
- 头部：标题 + 关闭按钮
- 生成中：旋转动画 + 精确进度条 + 百分比
- 预览区：
  - 左侧（flex-1）：简历模板预览（缩放 0.55 倍显示）
  - 右侧（260px）：
    - 模板选择（3 个选项卡片，选中态高亮）
    - 下载格式选择（PDF/Word 复选框）
    - 下载按钮
    - 重新生成按钮

#### 特性展示区（首页底部，分析前显示）
- 四列网格
- 每个特性卡片：
  - 渐变色图标容器（48x48）
  - 标题 + 描述
  - Hover：图标放大 1.1 倍

#### 页脚
- 居中灰色文字："ResumeAI — 基于AI的智能简历优化与面试准备平台"

### 2.2 简历模板预览（3 种风格）

- 成果列表：圆点符号，缩进排列
- 整体风格类似标准英文简历

#### 模板 B — 现代双栏
- 左栏（195px 宽）：深色渐变背景（#1a1a2e → #2d3748），白色文字
  - 姓名 + 个人标签
  - 联系方式（标签+值）
  - 技能特长（标签云样式）
  - 教育背景
  - 证书荣誉
- 右栏（flex-1）：白色背景
  - 个人摘要
  - 工作经历
  - 项目经历
  - 自我评价
- 左栏各模块标题：白色，底部 1px 半透明白色分隔线

#### 模板 C — 创意渐变
- 白色背景
- 顶部渐变色条（Indigo → Purple → Pink），圆角 12px，内含姓名+标签+联系信息

- 整体更有设计感，适合创意岗位

- 高度：8-10px

## 四、响应式设计

### 断点
- 移动端：< 768px
- 平板：768px - 1024px
- 桌面：> 1024px

### 适配规则
- 上传区域：桌面端 2 列，移动端 1 列
- 四维度指标卡片：桌面端 4 列，平板 2 列，移动端 2 列
- 优势/不足/建议：桌面端 3 列，移动端 1 列
- 模式选择：桌面端 3 列，移动端 1 列
- 后台管理：桌面端左右分栏，移动端上下堆叠
- 生成简历预览：桌面端预览+下载并排，移动端上下堆叠
- 进度条最大宽度：448px（max-w-md），居中
